# Security Policy

## Reporting

If you discover a security vulnerability, **do not open a public issue**.

Instead, contact the maintainer directly via GitHub's private vulnerability reporting: https://github.com/darkred1145/batomon-showdown-leaderboard-mod/security/advisories

## What to report

- PCK encryption key exposure
- Compromised credentials or tokens
- Any other sensitive data leaks

## Response

Reports will be acknowledged within 72 hours. We'll work with you to understand the impact and publish a fix.

## Disclosure

Please allow time for a fix before publicly disclosing the issue.
